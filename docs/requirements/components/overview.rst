Core Components Overview
========================

This section contains requirements for synarius-core subcomponents only.

Core component codes:

- ``CTL`` for controller
- ``MODEL`` for model
- ``CGEN`` for compiler/code generation
- ``RUN`` for run engine

Component requirement IDs must use:

- ``CORE-COMP-CTL-NNN``
- ``CORE-COMP-MODEL-NNN``
- ``CORE-COMP-CGEN-NNN``
- ``CORE-COMP-RUN-NNN``
